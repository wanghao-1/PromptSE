# PromptSE+: Multi-stage LLM-based Representation Learning for Drug Side Effect Prediction

## Data Files
- `drug_se_matrix.txt` – drug–side effect association matrix  
- `drugs.fpt` – drug molecular fingerprints  

## Feature Files
- `mols_vec.pkl` – molecular word embeddings generated by Mol2vec  
- `0403_3plus.xlsx` – side effect information vectors generated by Qwen‑plus3  
- `w2v.xlsx` – 200‑dimensional vectors encoding side effect terms using Word2Vec pre‑trained on a combined PubMed, PMC, and Wikipedia corpus  

## Code Files
- `model_PromptSE_plus.py` – PromptSE+ model definition  
- `train_PromptSE_plus.py` – program entry point  
- `utils.py` – data processing and validation methods  

## Model Parameters
- `model_params_0410.pth` – optimal parameters of the PromptSE+ model  
- `model_params.pth` – optimal parameters of the baseline model  

## Usage
```bash
python train_PromptSE_plus.py
